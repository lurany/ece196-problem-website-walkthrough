Checking General Open Recreation Hours

What works:

  • Gives a general idea of when the courts are available.

What is missing:

  • Doesn't provide the ability to check live occupancy.

Proposed Solution

  • Motion sensors located on each court scan for activity.
  • When a person walks near the sensor while using the court, the sensor sends a detection signal to the processor.
  • The processor records a timestamp when the signal is received and combines detections over time to estimate court activity.
  • The court activity data is sent to a live website where students can check whether courts are likely being used.

Testable Hypothesis

If PIR motion sensors are placed near UCSD campus courts and each motion event is recorded with a timestamp, then our system can estimate whether a court is active or inactive over time. This information can help students decide whether it is worth walking to the courts before they arrive.

We can test this by comparing the PIR sensor readings against real court observations. During testing, we would record when people are actively using the court and check whether our website shows similar activity patterns.

Brief Tutorial Pitch

PIR Motion Sensor Tutorial

This tutorial explains how a PIR motion sensor detects nearby movement by sensing changes in infrared radiation. Since people emit infrared radiation as body heat, the sensor can detect when someone moves within its range. It will cover the sensor's basic operation, its role in detecting court activity, and how its output signal can be used by a microcontroller in Triton CourtLive.

Object Recognition Tutorial

This tutorial allows the ESP32 dev board to detect and identify objects via a live camera sensor input. A YOLOV3 model is implemented using Convolutional Neural Network (CNN) image processing to successfully identify objects from the camera’s video data at around 2 FPS.

VU Meter Fan-Speed Controller

This tutorial is an extension of the VU meter mini project aims to control the speed of a fan based on claps by applying amplitude thresholding. One clap speeds it up and two claps slows it down.
